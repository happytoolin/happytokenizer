Encoding Display Test

This test verifies that the encoding display is working for non-GPT models.

Expected Encoding Display Names:

  • o200k_base should display as: "O200K Base (GPT-4o)"
  • cl100k_base should display as: "CL100K Base (GPT-4)"
  • llama3 should display as: "Llama 3"
  • deepseek should display as: "DeepSeek"
  • qwen should display as: "Qwen"
  • gemini should display as: "Gemini"
  • gemma should display as: "Gemma"
  • baichuan should display as: "Baichuan"

How to Test:

  1. Start the development server with pnpm dev
  2. Open http://localhost:4323 in your browser
  3. Select different non-GPT models from the dropdown (Llama 3, DeepSeek, Qwen, etc.)
  4. Check the "Encoding" field in the sidebar - it should show human-readable names
  5. Test both text and chat modes
